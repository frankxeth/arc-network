# Arc Network - Developer Tutorials

A collection of tutorials for building on Arc Testnet, a stablecoin-native Layer-1 blockchain built by Circle.

## 📁 Project Structure
## ✅ Completed Tutorials

### 1. Circle Entity Secret & Dev-Controlled Wallet
- Generate and register Entity Secret to Circle Console
- Create a Dev-Controlled Wallet on Arc Testnet

### 2. Deploy Smart Contract (HelloArchitect)
- Set up Hardhat + Solidity environment
- Deploy `HelloArchitect.sol` to Arc Testnet
- Interact with the contract (read & write)
- **Contract:** `0x1a2F5E6B63887855D829CD440a07f34AAAb44be6`

### 3. Register AI Agent (ERC-8004)
- Register an AI agent with onchain identity (ERC-721 NFT)
- Record reputation feedback from a validator wallet
- Request and verify KYC validation onchain
- **Explorer:** https://testnet.arcscan.app/address/0xE1769629A3C94Cf8d5dE5FBabCAF8B18944d810C

### 4. Monitor Contract Events *(in progress)*
- Set up a webhook endpoint
- Create an event monitor via Circle Contracts API
- Retrieve past event logs

## 🔧 Tech Stack
- Node.js / TypeScript
- Hardhat (smart contract development)
- Viem (Ethereum interactions)
- Circle Developer-Controlled Wallets SDK
- Arc Testnet
